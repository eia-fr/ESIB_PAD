Ce retard est dû au temps nécessaire au développement de toutes les interfaces graphiques à double.\section{Décision concernant la prochaine itération}Essayer de rattraper le retard pris lors de cette itération pour ne pas être pris de cours pour la fin de projet.\section{Conclusion}Cette $4^\textrm{ ème}$ itération s'est bien déroulée. Avec une interface graphique personnalisée telle que celle-ci, le temps pour produire une application compatible avec les 2 appareils est quasi le double de celui nécessaire pour une application compatible avec 1 seul appareil.

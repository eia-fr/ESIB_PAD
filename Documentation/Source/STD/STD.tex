Le deuxième niveau est celui faits dans le code et qui sont reproductive automatiquement(Unit test). Il est important d'indiquer que les cas de test sont imaginés en même temps que la spécification, ce qui nous permet d'avoir un point de référence concernant les objectif à atteindre.
